3.1 Authentication Service (auth.service.ts)
----------------------------------------------
Base URL: http://localhost:5006/api/v1/identity/auth

POST /register
- Request: { email, password, fullName }
- Response: ApiResponse with success flag
- Stores pending email in sessionStorage for verification flow

POST /verify-email-otp
- Request: { email, otp }
- Response: { accessToken, user } on success
- Handles auth success, stores token in sessionStorage

POST /resend-verification
- Request: { email }
- Response: ApiResponse

POST /forgot-password
- Request: { email }
- Response: ApiResponse
- Stores pending email for reset flow

POST /reset-password
- Request: { email, otp, newPassword }
- Response: ApiResponse

GET /users/me (via API_GATEWAY)
- Response: User profile data
- Used by: profile.component.ts

PUT /users/me (via API_GATEWAY)
- Request: { fullName }
- Updates user profile

PUT /users/me/password (via API_GATEWAY)
- Request: { currentPassword, newPassword }
- Handles password change

POST /login (via baseUrl)
- Request: { email, password }
- Response: { accessToken, user }
- On success: stores in sessionStorage, navigates to dashboard/admin
