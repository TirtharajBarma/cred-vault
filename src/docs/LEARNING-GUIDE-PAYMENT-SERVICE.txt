## What This Service Does

The Payment Service handles **payment processing with saga orchestration**. It's the most complex service because it coordinates multiple services in a single transaction.

**Capabilities:**
- Initiate payments (full or partial)
- OTP verification for payment security
- Saga orchestration for multi-step payments
- Payment status tracking
- Payment reversal (compensation)
- Transaction ledger

## What is Saga Orchestration?

Before diving into code, understand **why** we need sagas:

### The Problem
A payment involves multiple steps across multiple services:
1. Verify OTP
2. Update bill status (Billing Service)
3. Calculate rewards (Billing Service)
4. Deduct from card (Card Service)

If step 3 fails after step 2 succeeds, we have **inconsistent data**.

### The Solution: Saga with Compensation
If any step fails, we **reverse previous steps** (compensation):

```
Happy Path:
[A] → [B] → [C] → [D] → Done

Failure at C:
[A] → [B] → [C] → ✗
              ↓
         Compensate B
              ↓
         Compensate A
              ↓
         Failed
```

### How It's Implemented
1. State Machine tracks current state and completed steps
2. Each step publishes events to trigger next service
3. Next service publishes response event
4. State machine transitions based on response
5. If failure, triggers compensation events

## The Saga - The Heart of Payment Service

### PaymentOrchestrationSaga.cs (State Machine)

**Location:** `PaymentService.Application/Sagas/PaymentOrchestrationSaga.cs`

**What it is:** A MassTransit state machine that orchestrates the entire payment flow.

**Read this file VERY carefully** - it's the most complex part.

```csharp
public class PaymentOrchestrationSaga : MassTransitStateMachine<PaymentOrchestrationSagaState>
{
    public State AwaitingOtpVerification { get; private set; } = null!;
    public State AwaitingPaymentConfirmation { get; private set; } = null!;
    public State AwaitingBillUpdate { get; private set; } = null!;
    public State AwaitingRewardRedemption { get; private set; } = null!;
    public State AwaitingCardDeduction { get; private set; } = null!;
    public State Compensating { get; private set; } = null!;
    public State Completed { get; private set; } = null!;
    public State Failed { get; private set; } = null!;
    public State Compensated { get; private set; } = null!;
    
    public Event<IStartPaymentOrchestration> StartPayment { get; private set; } = null!;
    public Event<IOtpVerified> OtpVerified { get; private set; } = null!;
    public Event<IOtpVerificationFailed> OtpVerificationFailed { get; private set; } = null!;
    public Event<IPaymentProcessed> PaymentProcessed { get; private set; } = null!;
    public Event<IPaymentProcessFailed> PaymentProcessFailed { get; private set; } = null!;
    public Event<IBillUpdated> BillUpdated { get; private set; } = null!;
    public Event<IBillUpdateFailed> BillUpdateFailed { get; private set; } = null!;
    public Event<IRewardRedemptionCompleted> RewardRedemptionCompleted { get; private set; } = null!;
    public Event<IRewardRedemptionFailed> RewardRedemptionFailed { get; private set; } = null!;
    public Event<ICardDeducted> CardDeducted { get; private set; } = null!;
    public Event<ICardDeductionFailed> CardDeductionFailed { get; private set; } = null!;
    
    public PaymentOrchestrationSaga()
    {
        // Configure state instance
        InstanceState(x => x.CurrentState);
        
        // ==========================================
        // EVENT DEFINITIONS
        // ==========================================
        Event(() => StartPayment, 
            x => x.CorrelateById(m => m.Message.CorrelationId));
        
        Event(() => OtpVerified, 
            x => x.CorrelateById(m => m.Message.CorrelationId));
        
        // ... other events similarly configured
        
        // ==========================================
        // STATE MACHINE DEFINITIONS
        // ==========================================
        
        // Initial state → Start payment orchestration
        Initially(
            When(StartPayment)
                .Then(context =>
                {
                    // Store initial data in saga state
                    context.Saga.PaymentId = context.Message.PaymentId;
                    context.Saga.UserId = context.Message.UserId;
                    context.Saga.CardId = context.Message.CardId;
                    context.Saga.BillId = context.Message.BillId;
                    context.Saga.Amount = context.Message.Amount;
                    context.Saga.RewardsAmount = context.Message.RewardsAmount;
                    context.Saga.RewardsRedeemed = false;
                    context.Saga.OtpVerified = false;
                    context.Saga.PaymentProcessed = false;
                    context.Saga.BillUpdated = false;
                    context.Saga.CardDeducted = false;
                    context.Saga.CreatedAtUtc = DateTime.UtcNow;
                    context.Saga.UpdatedAtUtc = DateTime.UtcNow;
                })
                .TransitionTo(AwaitingOtpVerification)
                .Publish(context => new IRequestOtpVerification
                {
                    CorrelationId = context.Saga.CorrelationId,
                    UserId = context.Saga.UserId,
                    PaymentId = context.Saga.PaymentId
                })
        );
        
        // ==========================================
        // OTP VERIFICATION FLOW
        // ==========================================
        During(AwaitingOtpVerification,
            When(OtpVerified)
                .Then(context =>
                {
                    context.Saga.OtpVerified = true;
                    context.Saga.UpdatedAtUtc = DateTime.UtcNow;
                })
                .TransitionTo(AwaitingPaymentConfirmation)
                .Publish(context => new IStartPaymentProcess
                {
                    CorrelationId = context.Saga.CorrelationId,
                    PaymentId = context.Saga.PaymentId,
                    UserId = context.Saga.UserId,
                    CardId = context.Saga.CardId,
                    BillId = context.Saga.BillId,
                    Amount = context.Saga.Amount,
                    RewardsAmount = context.Saga.RewardsAmount
                }),
                
            When(OtpVerificationFailed)
                .Then(context =>
                {
                    context.Saga.PaymentError = context.Message.Error;
                    context.Saga.UpdatedAtUtc = DateTime.UtcNow;
                })
                .TransitionTo(Failed)
                .Publish(context => new IPaymentFailed
                {
                    CorrelationId = context.Saga.CorrelationId,
                    PaymentId = context.Saga.PaymentId,
                    Error = context.Saga.PaymentError ?? "OTP verification failed"
                })
        );
        
        // ==========================================
        // PAYMENT PROCESSING FLOW
        // ==========================================
        During(AwaitingPaymentConfirmation,
            When(PaymentProcessed)
                .Then(context =>
                {
                    context.Saga.PaymentProcessed = true;
                    context.Saga.UpdatedAtUtc = DateTime.UtcNow;
                })
                .TransitionTo(AwaitingBillUpdate)
                .Publish(context => new IStartBillUpdate
                {
                    CorrelationId = context.Saga.CorrelationId,
                    PaymentId = context.Saga.PaymentId,
                    BillId = context.Saga.BillId,
                    AmountPaid = context.Saga.Amount,
                    RewardsPoints = context.Saga.RewardsAmount / 0.25m // Convert to points
                }),
                
            When(PaymentProcessFailed)
                .Then(context =>
                {
                    context.Saga.PaymentError = context.Message.Error;
                    context.Saga.UpdatedAtUtc = DateTime.UtcNow;
                })
                .TransitionTo(Failed)
                .Publish(context => new IPaymentFailed
                {
                    CorrelationId = context.Saga.CorrelationId,
                    PaymentId = context.Saga.PaymentId,
                    Error = context.Saga.PaymentError ?? "Payment processing failed"
                })
        );
        
        // ==========================================
        // BILL UPDATE FLOW
        // ==========================================
        During(AwaitingBillUpdate,
            When(BillUpdated)
                .Then(context =>
                {
                    context.Saga.BillUpdated = true;
                    context.Saga.UpdatedAtUtc = DateTime.UtcNow;
                })
                .If(context => context.Saga.RewardsAmount > 0,
                    then => then
                        .TransitionTo(AwaitingRewardRedemption)
                        .Publish(context => new IStartRewardRedemption
                        {
                            CorrelationId = context.Saga.CorrelationId,
                            UserId = context.Saga.UserId,
                            BillId = context.Saga.BillId,
                            Amount = context.Saga.Amount,
                            RewardsAmount = context.Saga.RewardsAmount
                        })
                )
                .If(context => context.Saga.RewardsAmount <= 0,
                    then => then
                        .TransitionTo(AwaitingCardDeduction)
                        .Publish(context => new IStartCardDeduction
                        {
                            CorrelationId = context.Saga.CorrelationId,
                            CardId = context.Saga.CardId,
                            Amount = context.Saga.Amount,
                            PaymentId = context.Saga.PaymentId
                        })
                ),
                
            When(BillUpdateFailed)
                .Then(context =>
                {
                    context.Saga.BillUpdateError = context.Message.Error;
                    context.Saga.UpdatedAtUtc = DateTime.UtcNow;
                })
                .TransitionTo(Compensating)
                .Publish(context => new IStartCompensation
                {
                    CorrelationId = context.Saga.CorrelationId,
                    Reason = context.Message.Error,
                    StepsToCompensate = new List<string>()
                })
        );
        
        // ==========================================
        // REWARD REDEMPTION FLOW (SKIP IF NO REWARDS)
        // ==========================================
        During(AwaitingRewardRedemption,
            When(RewardRedemptionCompleted)
                .Then(context =>
                {
                    context.Saga.RewardsRedeemed = true;
                    context.Saga.UpdatedAtUtc = DateTime.UtcNow;
                })
                .TransitionTo(AwaitingCardDeduction)
                .Publish(context => new IStartCardDeduction
                {
                    CorrelationId = context.Saga.CorrelationId,
                    CardId = context.Saga.CardId,
                    Amount = context.Saga.Amount,
                    PaymentId = context.Saga.PaymentId
                }),
                
            When(RewardRedemptionFailed)
                .Then(context =>
                {
                    // Log but don't fail - rewards are optional
                    context.Saga.CompensationReason = context.Message.Error;
                    context.Saga.UpdatedAtUtc = DateTime.UtcNow;
                })
                // Continue without rewards
                .TransitionTo(AwaitingCardDeduction)
                .Publish(context => new IStartCardDeduction
                {
                    CorrelationId = context.Saga.CorrelationId,
                    CardId = context.Saga.CardId,
                    Amount = context.Saga.Amount,
                    PaymentId = context.Saga.PaymentId
                })
        );
        
        // ==========================================
        // CARD DEDUCTION FLOW
        // ==========================================
        During(AwaitingCardDeduction,
            When(CardDeducted)
                .Then(context =>
                {
                    context.Saga.CardDeducted = true;
                    context.Saga.UpdatedAtUtc = DateTime.UtcNow;
                })
                .TransitionTo(Completed)
                .Publish(context => new IPaymentCompleted
                {
                    CorrelationId = context.Saga.CorrelationId,
                    PaymentId = context.Saga.PaymentId,
                    UserId = context.Saga.UserId,
                    CardId = context.Saga.CardId,
                    BillId = context.Saga.BillId,
                    Amount = context.Saga.Amount,
                    RewardsApplied = context.Saga.RewardsRedeemed,
                    Timestamp = DateTime.UtcNow
                }),
                
            When(CardDeductionFailed)
                .Then(context =>
                {
                    context.Saga.CardDeductionError = context.Message.Error;
                    context.Saga.UpdatedAtUtc = DateTime.UtcNow;
                })
                .TransitionTo(Compensating)
                .Publish(context => new IStartCompensation
                {
                    CorrelationId = context.Saga.CorrelationId,
                    Reason = context.Message.Error,
                    BillUpdated = context.Saga.BillUpdated,
                    RewardsRedeemed = context.Saga.RewardsRedeemed
                })
        );
        
        // ==========================================
        // COMPENSATION FLOW
        // ==========================================
        During(Compensating,
            // If already compensated by downstream services
            When(RewardRedemptionCompleted)
                .TransitionTo(Compensated),
                
            // If compensation complete
            WhenEver(context => context.Saga.CurrentState == Compensating.Name)
                .Then(context =>
                {
                    context.Saga.CompensationAttempts++;
                    context.Saga.UpdatedAtUtc = DateTime.UtcNow;
                })
                // Additional compensation logic would go here
                .TransitionTo(Compensated)
                .Publish(context => new IPaymentFailed
                {
                    CorrelationId = context.Saga.CorrelationId,
                    PaymentId = context.Saga.PaymentId,
                    Error = context.Saga.CompensationReason ?? "Payment failed"
                })
        );
        
        // ==========================================
        // FINAL STATES
        // ==========================================
        During(Completed,
            When(Enter)
                .Then(context => 
                {
                    // Log completion
                })
        );
        
        During(Failed,
            When(Enter)
                .Then(context =>
                {
                    // Update payment status to Failed
                })
        );
    }
}
```

### 1. InitiatePaymentCommand & Handler

**Location:** `PaymentService.Application/Commands/InitiatePayment/InitiatePaymentCommandHandler.cs`

```csharp
public class InitiatePaymentCommandHandler : IRequestHandler<InitiatePaymentCommand, ApiResponse<PaymentInitiatedDto>>
{
    private readonly IPaymentRepository _paymentRepository;
    private readonly IBillServiceClient _billServiceClient;  // HTTP call to Billing
    private readonly IPublishEndpoint _publishEndpoint;
    
    public async Task<ApiResponse<PaymentInitiatedDto>> Handle(
        InitiatePaymentCommand request, 
        CancellationToken cancellationToken)
    {
        // STEP 1: Validate bill exists and belongs to user
        var bill = await _billServiceClient.GetBillAsync(request.BillId);
        if (bill == null)
        {
            return ApiResponse<PaymentInitiatedDto>.Fail("Bill not found");
        }
        
        if (bill.UserId != request.UserId)
        {
            return ApiResponse<PaymentInitiatedDto>.Fail("Unauthorized");
        }
        
        // STEP 2: Validate amount
        if (request.Amount <= 0)
        {
            return ApiResponse<PaymentInitiatedDto>.Fail("Invalid payment amount");
        }
        
        if (request.Amount > bill.Amount)
        {
            return ApiResponse<PaymentInitiatedDto>.Fail("Amount exceeds bill amount");
        }
        
        // STEP 3: Create payment record
        var otp = GenerateOtp();  // 6-digit OTP
        
        var payment = new Payment
        {
            Id = Guid.NewGuid(),
            UserId = request.UserId,
            CardId = request.CardId,
            BillId = request.BillId,
            Amount = request.Amount,
            PaymentType = request.Amount == bill.Amount 
                ? PaymentType.Full 
                : PaymentType.Partial,
            Status = PaymentStatus.Initiated,
            OtpCode = otp,
            OtpExpiresAtUtc = DateTime.UtcNow.AddMinutes(10),
            CreatedAtUtc = DateTime.UtcNow,
            UpdatedAtUtc = DateTime.UtcNow
        };
        
        await _paymentRepository.CreateAsync(payment);
        
        // STEP 4: Start saga
        var correlationId = Guid.NewGuid();
        
        await _publishEndpoint.Publish(new IStartPaymentOrchestration
        {
            CorrelationId = correlationId,
            PaymentId = payment.Id,
            UserId = payment.UserId,
            CardId = payment.CardId,
            BillId = payment.BillId,
            Amount = payment.Amount,
            RewardsAmount = request.RewardsToApply,
            RewardsRedeemed = false,
            Timestamp = DateTime.UtcNow
        });
        
        // STEP 5: Send OTP (via Notification Service)
        await _publishEndpoint.Publish(new IPaymentOtpGenerated
        {
            CorrelationId = correlationId,
            PaymentId = payment.Id,
            UserId = payment.UserId,
            OtpCode = otp,
            ExpiresAt = payment.OtpExpiresAtUtc!.Value,
            Timestamp = DateTime.UtcNow
        });
        
        return ApiResponse<PaymentInitiatedDto>.Success(new PaymentInitiatedDto
        {
            PaymentId = payment.Id,
            CorrelationId = correlationId,
            Amount = payment.Amount,
            OtpRequired = true,
            ExpiresAt = payment.OtpExpiresAtUtc!.Value
        }, "Payment initiated. Please verify with OTP.");
    }
}
```

### 2. VerifyOtpCommand & Handler

**Location:** `PaymentService.Application/Commands/VerifyOtp/VerifyOtpCommandHandler.cs`

```csharp
public class VerifyOtpCommandHandler : IRequestHandler<VerifyOtpCommand, ApiResponse<bool>>
{
    private readonly IPaymentRepository _paymentRepository;
    private readonly IPublishEndpoint _publishEndpoint;
    
    public async Task<ApiResponse<bool>> Handle(
        VerifyOtpCommand request, 
        CancellationToken cancellationToken)
    {
        var payment = await _paymentRepository.GetByIdAsync(request.PaymentId);
        
        if (payment == null)
        {
            return ApiResponse<bool>.Fail("Payment not found");
        }
        
        if (payment.UserId != request.UserId)
        {
            return ApiResponse<bool>.Fail("Unauthorized");
        }
        
        if (payment.Status != PaymentStatus.Initiated)
        {
            return ApiResponse<bool>.Fail("Payment is not in valid state");
        }
        
        // Check OTP expiry
        if (payment.OtpExpiresAtUtc < DateTime.UtcNow)
        {
            return ApiResponse<bool>.Fail("OTP has expired");
        }
        
        // Verify OTP
        if (payment.OtpCode != request.Otp)
        {
            return ApiResponse<bool>.Fail("Invalid OTP");
        }
        
        // OTP verified - publish event for saga
        await _publishEndpoint.Publish(new IOtpVerified
        {
            CorrelationId = request.CorrelationId,
            PaymentId = payment.Id,
            Timestamp = DateTime.UtcNow
        });
        
        return ApiResponse<bool>.Success(true, "OTP verified successfully");
    }
}
```

## Infrastructure Layer - Understanding Each File

### 1. PaymentDbContext.cs

**Location:** `PaymentService.Infrastructure/Persistence/Sql/PaymentDbContext.cs`

**Key Point:** Configures saga state for MassTransit.

```csharp
public class PaymentDbContext : DbContext
{
    public DbSet<Payment> Payments { get; set; }
    public DbSet<Transaction> Transactions { get; set; }
    public DbSet<PaymentOrchestrationSagaState> SagaStates { get; set; }
    
    protected override void OnModelCreating(ModelBuilder modelBuilder)
    {
        base.OnModelCreating(modelBuilder);
        
        // Payment configuration
        modelBuilder.Entity<Payment>(entity =>
        {
            entity.ToTable("payments");
            entity.HasKey(e => e.Id);
            entity.HasIndex(e => e.UserId);
            entity.HasIndex(e => e.BillId);
            entity.Property(e => e.Status).HasConversion<int>();
            entity.Property(e => e.PaymentType).HasConversion<int>();
        });
        
        // Transaction configuration
        modelBuilder.Entity<Transaction>(entity =>
        {
            entity.ToTable("transactions");
            entity.HasKey(e => e.Id);
            entity.HasIndex(e => e.PaymentId);
            entity.HasIndex(e => e.UserId);
            entity.Property(e => e.Type).HasConversion<int>();
        });
        
        // Saga state configuration - MassTransit requires specific setup
        modelBuilder.Entity<PaymentOrchestrationSagaState>(entity =>
        {
            entity.ToTable("payment_orchestration_saga_state");
            
            // CorrelationId is both PK and index
            entity.HasKey(e => e.CorrelationId);
            
            // Index for state queries
            entity.HasIndex(e => e.CurrentState);
            
            // Configure string lengths
            entity.Property(e => e.CurrentState).HasMaxLength(64);
        });
    }
}
```

#### PaymentCompletedConsumer.cs

**Location:** `PaymentService.Infrastructure/Consumers/PaymentCompletedConsumer.cs`

**What it does:** When saga completes, update payment and create ledger entry.

```csharp
public class PaymentCompletedConsumer : IConsumer<IPaymentCompleted>
{
    private readonly IPaymentRepository _paymentRepository;
    private readonly ITransactionRepository _transactionRepository;
    private readonly IPublishEndpoint _publishEndpoint;
    
    public async Task Consume(ConsumeContext<IPaymentCompleted> context)
    {
        var message = context.Message;
        
        // Update payment status
        var payment = await _paymentRepository.GetByIdAsync(message.PaymentId);
        if (payment != null)
        {
            payment.Status = PaymentStatus.Completed;
            payment.UpdatedAtUtc = DateTime.UtcNow;
            await _paymentRepository.UpdateAsync(payment);
        }
        
        // Create ledger entry
        var transaction = new Transaction
        {
            Id = Guid.NewGuid(),
            PaymentId = message.PaymentId,
            UserId = message.UserId,
            Amount = message.Amount,
            Type = TransactionType.Payment,
            Description = $"Payment for bill {message.BillId}",
            CreatedAtUtc = DateTime.UtcNow
        };
        
        await _transactionRepository.CreateAsync(transaction);
        
        // Publish for notification
        await _publishEndpoint.Publish(new IPaymentCompleted
        {
            PaymentId = payment!.Id,
            UserId = payment.UserId,
            Amount = payment.Amount,
            CardLast4 = "1234",  // Get from card service if needed
            Timestamp = DateTime.UtcNow
        });
    }
}
```

```csharp
public class RewardRedemptionConsumer : IConsumer<IStartRewardRedemption>
{
    private readonly IRewardServiceClient _rewardServiceClient;  // HTTP to Billing
    private readonly IPublishEndpoint _publishEndpoint;
    
    public async Task Consume(ConsumeContext<IStartRewardRedemption> context)
    {
        var message = context.Message;
        
        try
        {
            // Call Billing Service to redeem rewards
            var result = await _rewardServiceClient.RedeemRewardsAsync(new RedeemRewardsRequest
            {
                UserId = message.UserId,
                BillId = message.BillId,
                PointsToRedeem = message.RewardsAmount / 0.25m  // Convert to points
            });
            
            if (result.Success)
            {
                await _publishEndpoint.Publish(new IRewardRedemptionCompleted
                {
                    CorrelationId = message.CorrelationId,
                    UserId = message.UserId,
                    PointsRedeemed = result.Data.PointsRedeemed,
                    MonetaryValue = result.Data.MonetaryValue,
                    Timestamp = DateTime.UtcNow
                });
            }
            else
            {
                await _publishEndpoint.Publish(new IRewardRedemptionFailed
                {
                    CorrelationId = message.CorrelationId,
                    UserId = message.UserId,
                    Error = result.Message,
                    Timestamp = DateTime.UtcNow
                });
            }
        }
        catch (Exception ex)
        {
            await _publishEndpoint.Publish(new IRewardRedemptionFailed
            {
                CorrelationId = message.CorrelationId,
                UserId = message.UserId,
                Error = ex.Message,
                Timestamp = DateTime.UtcNow
            });
        }
    }
}
```

## Saga States Explained

| State | Meaning |
|-------|---------|
| Initial | Saga just started |
| AwaitingOtpVerification | Waiting for user to verify OTP |
| AwaitingPaymentConfirmation | OTP verified, verifying payment details |
| AwaitingBillUpdate | Updating bill status |
| AwaitingRewardRedemption | Redeeming rewards (if any) |
| AwaitingCardDeduction | Deducting from card balance |
| Compensating | Rolling back completed steps |
| Completed | All steps successful |
| Failed | Payment failed |
| Compensated | Compensation complete |

## Exercise: Trace Failure at Card Deduction

1. Steps 1-5 complete successfully
2. Step 6: Card Service returns `ICardDeductionFailed`
3. Saga transitions to `Compensating` state
4. Publishes `IStartCompensation` with:
   - `BillUpdated = true` → needs rollback
   - `RewardsRedeemed = true` → needs rollback
5. Compensation events published:
   - `IRevertBill` → Billing Service
   - `IRevertRewardRedemption` → Billing Service
6. Downstream services revert their changes
7. Saga transitions to `Compensated`
8. `IPaymentFailed` published for notification
